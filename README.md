# Name of the Project

Alstom Project 1

## Pitch of the project

Dans les wagons ALSTOM, les opérateurs doivent assembler des dizaines de composants isolants répartis en quatre zones : paroi droite, paroi gauche, plafond et avant/sol.

Chaque zone contient un sac séparé contenant une cinquantaine de pièces, dont beaucoup semblent presque identiques.

Une perte de temps importante se produit avant même le début de l'assemblage : les ouvriers doivent fouiller dans les quatre sacs pour trouver celui qui contient les pièces essentielles à installer en priorité.

Une fois la pièce trouvée, une deuxième source d'inefficacité apparaît : localiser son emplacement exact sur le plan papier 2D très complexe.

Avec des dizaines de codes et de numéros, le plan est difficile à lire, en particulier pour les nouveaux arrivants ou les opérateurs ayant des difficultés de lecture.

Ces deux étapes combinées ralentissent le flux de travail et augmentent la dépendance vis-à-vis des travailleurs expérimentés.

Notre solution est une application pour tablette conçue pour éliminer ces deux difficultés.
La boîte contenant les pièce prioritaires est marquée d'un autocollant de couleur et les pièces essentielles sont également marquées d'un autocollant de couleur, ce qui permet aux opérateurs de les identifier et de les installer instantanément.

Pour toutes les autres pièces, le scan de la pièce affiche son emplacement exact à l'intérieur d'un modèle 3D du wagon, ce qui évite d'avoir à interpréter le plan en 2D.
Cela leur permet d'installer n'importe quelle pièce sans perdre de temps à chercher son emplacement.

L'application comprend également un tutoriel vidéo simple et très visuel qui montre étape par étape l'ensemble du processus de préparation et d'installation, rendant la formation beaucoup plus facile et accessible.

Cela permet une installation plus rapide et un transfert de connaissances plus fluide entre les collaborateurs.
